## Why AI Model Test Bench?

Testing local LLMs is messy.

Different prompts, inconsistent parameters, and no clear way to compare results.

**AI Model Test Bench gives you a structured environment to evaluate AI models scientifically.**

Run standardized prompt suites, measure performance, and rank models — all locally with **no cloud dependencies**.

---

## ✨ Key Highlights

⚡ **Benchmark Local Models**  
Run structured test suites across multiple GGUF models.

📊 **Visual Leaderboards**  
Compare quality, speed, and efficiency with an interactive dashboard.

🧠 **Built-in Prompt Suites**  
Reasoning, coding, creative writing, factual recall, and safety.

🔧 **Full Parameter Control**  
Temperature, top-p, context window, GPU layers, mirostat, and more.

📡 **Live Metrics Streaming**  
Tokens/sec, TTFT, RAM, and VRAM tracked in real time.

📁 **Exportable Results**  
Download test results as CSV or JSON.
